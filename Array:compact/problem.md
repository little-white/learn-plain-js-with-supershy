> 来自lodash中文官网的[compact](http://lodashjs.com/docs/#_compactarray)

创建一个新数组并包含原数组中所有的非假值元素。例如 `false`、`null`、 `0`、`""`、`undefined` 和 `NaN` 都是“假值”。

#### 参数

1. `array` *(Array)*: 数组参数.

#### 返回值

*(Array)*: 返回过滤假值后的数组.

#### 例子

```javascript
_.compact([0, 1, false, 2, '', 3]);
// => [1, 2, 3]
```
